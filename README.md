## ステップ1️⃣：Node.jsをインストール（初回のみ・10分）

### 確認方法（すでにある場合はスキップ）
1. Windows + R を押す
2. 「cmd」と入力してEnter
3. 黒い画面で `node -v` と入力してEnter
4. バージョンが表示されたら → ステップ2へ
5. エラーが出たら → 以下の手順でインストール

### インストール手順
1. ブラウザで **https://nodejs.org/** を開く
2. **緑色の大きいボタン**（LTS版・推奨版）をクリック
3. ダウンロードされた **「node-...msi」** ファイルをダブルクリック
4. インストーラーが開いたら **「Next」を連打**
5. 完了！

---

## ステップ2️⃣：このフォルダをデスクトップに置く（1分）

1. この **aicomp-webapp-new** フォルダ全体をデスクトップにコピー
2. フォルダ名は **「aicomp-app」** に変更してもOK

---

## ステップ3️⃣：アプリを起動（5分）

### A. フォルダを開く
デスクトップの **aicomp-app** フォルダをダブルクリック

### B. コマンドプロンプトを開く（超重要！）

**方法：フォルダの中でコマンドプロンプトを開く**

1. フォルダの上部にあるアドレスバー（パスが表示されてる場所）をクリック
2. 全選択されて青くなる
3. **「cmd」** と入力してEnter
4. 黒い画面が開く ← これが正解！

### C. 必要なファイルをインストール（初回のみ）

黒い画面に以下をコピー＆ペーストしてEnter：
```
npm install
```

⏳ **1-2分待つ**（いろいろダウンロードされます）

「added XX packages」と表示されたら完了！✅

### D. アプリを起動！

黒い画面に以下をコピー＆ペーストしてEnter：
```
npm start
```

🎉 **「AI COMP 管理システムサーバーが起動しました！」** と表示されたら成功！

## 💡 使い方

### ✅ 新規契約を登録
1. **「➕ 新規契約登録」** ボタンをクリック
2. フォームに情報を入力
3. プランを選択すると料金が自動入力される
4. **「✅ 登録する」** をクリック

### 🔍 検索
1. 検索ボックスに会社名や担当者名を入力
2. リアルタイムで絞り込まれる

### ✏️ 編集
1. テーブルの **「編集」** ボタンをクリック
2. 情報を修正して保存

### 🗑️ 削除
1. テーブルの **「削除」** ボタンをクリック
2. 確認して削除

### 📤 CSVエクスポート
1. **「📤 CSVエクスポート」** ボタンをクリック
2. CSVファイルがダウンロードされる

## ❓ トラブルシューティング

### 「npm が見つかりません」と出る
→ Node.jsがインストールされていません
→ ステップ1からやり直してください

### 「ポートが使用されています」と出る
→ 別のアプリが3000番を使っています
→ 黒い画面で以下を実行：
```
set PORT=3001 && npm start
```
→ ブラウザで `http://localhost:3001` を開く

### 画面が真っ白
→ サーバーが起動しているか確認
→ URLが `http://localhost:3000` で正しいか確認
→ Ctrl+F5でページを更新

### エラーメッセージが出る
→ エラー内容を教えてください
→ 一緒に解決します！

## 📞 よくある質問

**Q: インターネット接続は必要ですか？**
A: 初回の「npm install」の時だけ必要です。その後はオフラインで使えます。

**Q: 他のパソコンでも使えますか？**
A: はい！このフォルダを他のPCにコピーして、同じ手順で起動できます。

**Q: データは消えませんか？**
A: contracts.jsonにすべて保存されるので、サーバーを止めても消えません。

**Q: 複数人で同時に使えますか？**
A: このままでは1台のPC内だけです。複数人で使うにはサーバー設定が必要です。

**Q: Excelにデータを移せますか？**
A: はい！「CSVエクスポート」でCSVファイルをダウンロードして、Excelで開けます。
